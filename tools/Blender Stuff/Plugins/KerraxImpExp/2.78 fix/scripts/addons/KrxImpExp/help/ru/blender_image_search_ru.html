Поиск изображений

Вы можете помочь Blender'у с поиском изображений, указав папки, в которых их следует искать. Чтобы сделать это, выберите "User Preferences" из главного меню (см. рисунок ниже).

Следующее окно появится на экране. Нажмите кнопку "Add a path with subpathes".

Далее будет показан диалог как для открытия файла. Найдите корневую папку для текстур и нажмите кнопку "Select path". Будет использован только путь к каталогу. Имя файла ("untitled.blend" на рисунке ниже) не будет использоваться.

Если все правильно, то теперь вы видите список всех папок, которые будут использоваться для поиска изображений, когда вы импортируете любые .3ds или .asc -файлы.

Если вы хотите, то вы можете редактировать этот список папок с помощью других кнопок.
